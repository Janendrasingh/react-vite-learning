# 🚀 React + Vite First Learning Project

This repository is my **first hands-on learning project** using modern frontend tools like **React.js** and **Vite**.

It represents my journey from zero to understanding how a real-world React project is initialized, structured, and run using npm and Vite.

---

## 📌 About This Project

This is a **beginner-level learning repository**, created to understand:

- How modern React projects are created using Vite
- What happens when we run `npm create vite@latest`
- How npm initializes a project
- How dependencies are installed and managed
- How React components work
- How development servers run locally

This is NOT a production project — it is purely for **learning and experimentation**.
